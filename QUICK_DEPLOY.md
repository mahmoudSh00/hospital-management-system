# دليل النشر السريع 🚀

## الطريقة الأسرع: GitHub Pages

### 1. إنشاء حساب GitHub
- اذهب إلى: https://github.com
- انقر "Sign up" وأنشئ حساب

### 2. إنشاء مستودع جديد
- انقر الرمز "+" ثم "New repository"
- اسم المستودع: `hospital-management-system`
- اجعله **Public**
- انقر "Create repository"

### 3. رفع الملفات
- انقر "uploading an existing file"
- اسحب جميع ملفات المشروع
- اكتب: "Initial commit"
- انقر "Commit changes"

### 4. تفعيل GitHub Pages
- اذهب إلى "Settings"
- انزل إلى "Pages"
- اختر "Deploy from a branch"
- اختر "main" و "/ (root)"
- انقر "Save"

### 5. الرابط النهائي
`https://YOUR_USERNAME.github.io/hospital-management-system`

## بدائل أخرى سريعة:

### Netlify (الأسهل)
1. اذهب إلى: https://netlify.com
2. سجل دخول بـ GitHub
3. اسحب مجلد المشروع إلى الصفحة
4. انتظر النشر!

### Vercel
1. اذهب إلى: https://vercel.com
2. سجل دخول بـ GitHub
3. انقر "New Project"
4. اختر المستودع
5. انقر "Deploy"
